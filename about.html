About Cocktailopedia

Cocktailopedia is a web application built by Travis S, Aaron F, and Jason B. A user is initially presented with a random "Cocktail of the Day" that can clicked on to view the ingredients and directions to make it. On the top of the page, the user can select the find tab and find the dropdown menu for types of alcohols. The user can select an alcohol type and be presented with many possibilities for cocktails they may want. Any of these drinks are able to be saved for future needs.

Enjoy!
